Проект: "Система интеллектуального планирования накоплений для студентов с целью прогнозирования даты достижения цели на основе динамики  доходов и расходов"

  1. Концепция проекта
Создание веб-сервиса для финансового планирования, ориентированного на студентов. В отличие от простых журналов расходов, данный сервис реализует логику динамического распределения остатка: система анализирует текущий баланс пользователя и автоматически распределяет его между всеми активными целями, рассчитывая реалистичный прогноз их достижения на основе текущего темпа накоплений.

  2. Роли пользователей
- Авторизованный пользователь: Ведение личного финансового профиля, фиксация доходов и расходов через формы, создание финансовых целей (до 3-х одновременно) и получение автоматических расчетов прогресса.
- Администратор (Django Admin): Управление справочником категорий и контроль базы данных.

  3. Модели данных
3.1 Category (Категория): Типизация операций (например: «Стипендия», «Продукты», «Транспорт»). Поля: name, is_income (тип потока).
3.2 FinancialGoal (Финансовая цель): Объект сбора средств. Поля: user, title, target_amount.
3.3 Transaction (Транзакция): Финансовая операция. Поля: user, category, amount, date (автоматическая фиксация времени), description.

  4. Ключевой функционал
- Учет операций: Ввод данных через Django ModelForms. Расчет чистого баланса в реальном времени.
- Параллельное накопление: Алгоритм делит общий остаток средств пользователя поровну между всеми созданными целями, имитируя стратегию распределения по конвертам.
- Интеллектуальный прогноз: На основе даты первой транзакции система вычисляет средний ежедневный прирост капитала (daily_pace) и прогнозирует конкретную дату исполнения каждой цели.
- Валидация: Ограничение на создание более 3-х целей для фокусировки внимания пользователя.

  5. Визуализация и стек технологий
- Backend: Python / Django
- Визуализация: Построение круговой диаграммы с использованием библиотеки Matplotlib. График отображает структуру бюджета (доходы vs расходы), позволяя оценить финансовую устойчивость.
- Frontend: Адаптивный интерфейс на базе Tailwind CSS.
