  Smart Planner — Система финансового прогнозирования
Веб-сервис для студентов, который не просто фиксирует расходы, а рассчитывает дату достижения цели на основе реального темпа накоплений.

  Ссылка на сайт
https://web-production-37cd8.up.railway.app/

  Основная идея
Проблема большинства трекеров — они показывают прошлое. Smart Planner сфокусирован на будущем. Система использует алгоритм распределения остатка, который виртуально делит ваш текущий баланс между всеми целями и прогнозирует, когда вы сможете их реализовать.

  Стек технологий
Backend: Python, Django
Frontend: HTML5, Tailwind CSS
Аналитика и графики: Matplotlib
Deploy: Railway

  Ключевые функции
Интеллектуальный прогноз: Автоматический расчет ежедневного темпа накоплений (daily_pace) с момента первой транзакции.
Визуализация: Круговая диаграмма соотношения доходов и расходов.
Лимиты: Ограничение до 3-х активных целей для эффективного финансового фокуса.

  Структура проекта
wallets/models.py — Схема данных (Category, Transaction, FinancialGoal).
wallets/views.py — Бизнес-логика, расчет прогнозов и темпа.
wallets/utils.py — Логика построения графиков через Matplotlib.
wallets/templates — HTML-страницы.
